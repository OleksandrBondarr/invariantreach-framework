# InvariantReach — AI-Assisted Smart Contract Audit Framework

InvariantReach is a structured invariant-based smart contract audit methodology
combining multi-model reasoning, strict reachability classification (R/A/U),
and meta-level adjudication.

## Core Principles

- Invariant-first analysis
- R / A / U reachability labeling
- CALL vs SUCCESS separation
- Strict vs Practical risk overlay
- Multi-model cross-verification

⚠ Detailed orchestration logic and model coordination layers are proprietary.
